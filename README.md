# Constructora Infraestructura

Este proyecto consistió en el desarrollo de un sitio web para la empresa Constructora Infraestructura. El objetivo principal fue proporcionar presencia en línea efectiva y profesional para el negocio.

## Descripción

Constructora Infraestructura es una empresa dedicada a servicios de construcción. El sitio web desarrollado para ellos incluye características destacadas como diseño profesional, navegación intuitiva de una sola página (landing page) con diversas secciones entre las que destaca su galería de proyectos así como su formulario de contacto.

## Características Destacadas

- **Diseño Profesional:** El sitio web cuenta con un diseño que refleja seriedad y profesionalismo, adecuado al branding actual de la empresa.
- **Navegación Intuitiva:** Se ha diseñado una interfaz de usuario fácil de navegar, con toda la información relevante accesible desde una sola página.
- **Sección Proyectos:** Para atraer a potenciales clientes, se incluye una sección con los proyectos de la empresa agrupados por ciudades, esto es un mapa utilizando Leaflet por sus cualidades Open Source y ventanas modales.
- **Formulario de Contacto:** Se proporciona un formulario de contacto donde los usuarios pueden comunicarse fácilmente con la empresa para consultas o contratación de servicios.

## Tecnologías Utilizadas

- HTML 5
- CSS
- JavaScript
- Bootstrap (como framework de desarrollo front-end)
- SwiperJS
- Leaflet
- AOS: Animated On Scroll
- ReCaptcha de Google
